## 📚 Sobre o Repositório

Este repositório foi criado com o objetivo de organizar e compartilhar mais de **500 exercícios de programação** desenvolvidos ao longo da graduação em Ciência da Computação.

O conteúdo é focado em:

- Algoritmos
- Estruturas de Dados
- Lógica de Programação
- Resolução de Problemas
- Pensamento Computacional

O objetivo principal é fortalecer a base da programação e também auxiliar estudantes dos primeiros períodos de Ciência da Computação e áreas relacionadas.
